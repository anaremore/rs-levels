# Install Draft

RS Levels is not packaged yet. The first development release should support:

- Windows
- macOS
- Linux
- Chromium-based browsers that support Manifest V3 extensions

## Planned Developer Flow

```powershell
git clone git@github.com:anaremore/rs-levels.git
cd rs-levels
```

Then:

1. Start the local service.
2. Load the browser extension unpacked.
3. Open RocketScooter in the browser.
4. Read `http://127.0.0.1:8765/health`.

Exact commands will be added when implementation begins.
